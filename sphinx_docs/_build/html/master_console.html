master_console module¶

class master_console.MasterConsole(listen_port=65000)¶

Bases: object

Class for handling all user interaction with the pi.

borrow_books(userID, username)¶

Asks user for the id of the book they wish to borrow, creates a borrow event in database and creates a calendar event for the borrowed book.

Args:

userID (str): ID of the user borrowing the book username (str): Username of the user borrowing the book

connect_to_reception()¶

Waits for a connection to a reception pi and then waits until a username is sent where it will let the display_console method handle the user

display_console(userID, username, first_name)¶

Displays console menu and gets an option from user.

Args:

username (str): Username of the connected user.

return_books()¶

Asks user for the ID of the book they wish to return, updates the corresponding borrow database entry and deletes the corresponding calendar event.

search_books()¶

Asks user to specify a property and property value, which is then used in a search of all books in the database and the result is formatted and displayed to the user

valid_date(date)¶

Checks if given date is valid and returns true if so, false if not.

Args:

date (str): String of date in the format of YYYY-MM-DD.

Returns:

Whether or not the date is valid.
