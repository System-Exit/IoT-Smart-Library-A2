main_console module¶

class main_console.ReceptionConsole¶

Bases: object

Class for handling user information and user login for connecting to the ‘Master Pi’

credential_login()¶

Prompt user for their login details, scan database for username, compare hashed password.

display_console()¶

Displays the console and presents the user options to login/register

face_login()¶

Allow user to login via facial recognition if they have registered their face for facial recognition.

new_user_details()¶

Prompt user for information including their first name, last name and email, then allow them to set a username and password for creating a new account.
